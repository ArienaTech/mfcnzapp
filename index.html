Miss Five Crowns App
First in New Zealand

The simple, intuitive app that allows user to vote and manage their favorite contestants.

Learn More

Goal of the Final App

This application introduces the idea of a Voting Application for a Beauty Pageant; designed to allow user to be able to vote for their favourite candidates and adding their candidates to their favourite lists with a larger mission to keep user up to date with the competition, educate user of the chosen charity, and allow user to be involve in the judging process.

User can register and sign in via Facebook, Email and Twitter. They also have the options to connect as a guest. User will get access to complete profiles of all candidates; personalâ€™s bios, official photos, charity events. The app allows user to filter search for region and contestant name.

A primary persona for this application is Pam Cummings, a founding director at Premiere International Ltd, who managed Miss Five Crowns New Zealand (MFCNZ) and plans to expand the pageant to a larger scale. MFCNZ currently have a website that does not have a voting system for its user. Check Out MFCNZ website!
