Apps Comparison

Comparison with existing apps, MFCNZ and Miss U App (Miss Universe App)

MFCNZ App

Advantages

  • Votes for favorite contestants and add their favorite contestants to their list.
  • Stay connected with all the actions around Miss Five Crowns NZ.
  • Get access to MFCNZ story and contestant bios and profile.
  • The user can search contestants by name and region.
  • MFCNZ app can be accessible in many platforms and devices; Android, iOS, Windows Phone and Blackberry as the app will be developed using jQuery Mobile framework that can create cross-platform mobile app.



Disadvantages

  • An app developed in JQuery might perform slower on the Android operating system and Chrome but run faster on IOS devices. JQuery applications cannot outperform native mobile application in the category of speed.

Miss Universe App

Advantages

  • Votes for favorite contestants, play exclusive games and watch live shows.
  • Stay connected with all the action around Miss Universe, Miss USA and Miss Teen USA. User only needs to download this app to keep up to date with these pageants.
  • Get access to Miss Universe story and contestant bios, profile and videos.
  • User can use the advanced search option to find contestants by year, name or state.
  • Miss Universe app is accessible in both Android and IOS platforms.
  • User can check out the calendar of events and vote for their favorite contestant before and during the competitions.

Disadvantages

  • The app might be slower compare to the Miss Five Crowns app that only managed one pageant. The Miss Universe app used a bigger database and it takes time for it to load.
